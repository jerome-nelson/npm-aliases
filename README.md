
# npm-aliases

  Commonly used npm aliases. Fork for more!

## Installation

```bash
$ source aliases.sh
```

## Aliases

| Alias | Command |
|-------|---------|
| `ni`  | `npm install` |
| `nid` | `npm install --save-dev` |
| `nig` | `npm install -g` |
| `nt`  | `npm test` |
| `nit` | `npm install && npm test` |
| `nr`  | `npm run-script` |
| `nl`  | `npm link` |
